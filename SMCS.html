STADIUM MANAGEMENT AND CONTROL SYSTEM (SMCS)

Stadium management and control system was developed to improve the working principle of Asmara stadium i.e. to make all the manual work computerized and it was expected to minimize the time consumed and the man power used by the system. Stadium management and control system(SMCS) as its name suggests it is all about managing the activities done inside the stadium and controlling them. While thinking of developing SMCS, we didn’t have much idea on how to implement, since it was our first time to implement such kind of application. But we ahead with it .we faced several problems but we have learned a lot from those challenges and able to gain much knowledge from them. Our project consists of website and small application software. Like the other entire website the website contains menu bars with different links associated with it. Most importantly our website provides a registration form to users so that the users can reserve seats through online reservation system with no chance of wasting time. The application software calculates the final result by retrieving data from the database. The administrator has full control over this software and can update, delete and modify information. There are three buttons the search button, finance button and statistics button. This helps to save the time spent in calculating this money and eliminates errors in handling it. Some of the features of the new website and application software are:

  • User friendliness
  • Easy search facility
  • Less error
  • Quick registration
  • Reliable and secured I.T. (Information Technology) system and etc…
  • Easy record manipulation
